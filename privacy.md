## 我們蒐集什麼

本站由 GitHub Pages 託管，屬於靜態網站，**不主動蒐集個人識別資料**（PII）。在你造訪本站時，下列資料可能被被動紀錄：

### 1. 伺服器存取紀錄（GitHub Pages）
GitHub 在提供靜態託管服務時會紀錄 IP 位址、User-Agent、referrer 等，用於服務穩定與濫用防護。詳見 [GitHub 隱私權聲明](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement)。

### 2. 網站流量分析（Google Analytics 4）
若已啟用，本站使用 [Google Analytics 4](https://analytics.google.com/) 蒐集匿名化的使用資料，包含：

- 造訪頁面、停留時間、進出點
- 裝置類型、瀏覽器、作業系統
- 大致地理位置（國家 / 城市，由 IP 推算，不儲存完整 IP）
- referrer 來源網站

這些資料**不用於識別個人身分**，僅用於了解讀者分布與文章反應，協助決定寫作方向。

### 3. Cookies
- **必要 cookies**：無。
- **分析 cookies**：Google Analytics 使用 `_ga`、`_ga_<id>` cookies 追蹤不重複訪客與 session，保存期限最長 2 年。

你可以在瀏覽器設定中封鎖 cookies，或安裝 [Google Analytics Opt-out 擴充套件](https://tools.google.com/dlpage/gaoptout) 拒絕被追蹤。

### 4. 廣告（未來可能啟用）
若日後本站啟用 Google AdSense 或類似廣告服務，第三方廣告商可能會使用 cookies 與類似技術蒐集非個人識別資料，以投放個人化廣告。屆時本頁會更新揭露。你可於 [Google 廣告設定](https://adssettings.google.com/) 管理廣告個人化。

## 資料保留

- 本站自身不儲存任何訪客資料。
- GitHub Pages 存取紀錄依 GitHub 政策保留。
- Google Analytics 資料依 GA 設定保留（預設 14 個月）。

## 你的權利

依據 GDPR、台灣個人資料保護法等規範，你有權：

- 拒絕被追蹤（封鎖 cookies / 使用 opt-out）
- 向 Google 申請刪除你的分析資料

本站文章大量引用 GitHub、第三方部落格、社群媒體等外部連結。點擊後將離開本站，適用各該網站之隱私政策，本站不負連帶責任。
