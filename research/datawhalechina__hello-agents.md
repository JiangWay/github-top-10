# [datawhalechina/hello-agents](https://github.com/datawhalechina/hello-agents) — 深度研究

## 深度研究（2026-05-10 首次）

### 專案定位
[datawhalechina/hello-agents](https://github.com/datawhalechina/hello-agents) 是中國開源 ML 教育社群 Datawhale 推出的《從零開始構建智能體》系統化教程，主打把學習者「從 LLM 使用者升級為 Agent 系統建構者」。截至本日 stars 約 45.6k、fork 5.5k，貢獻者以 [jjyaoao](https://github.com/jjyaoao) 為核心（464 commits），總計 30+ 名社群協作者。已上 GitHub Trending、官網 <https://hello-agents.datawhale.cc>。

### 核心架構 / 主要概念
教程共 16 章、5 大區塊：
- **基礎篇** — Agent 演化與 LLM 局限
- **構建篇** — 實作 ReAct / Plan-and-Solve / Reflection 等典型範式，並比較 Dify、Coze、n8n、AutoGen、AgentScope、LangGraph
- **進階篇** — Memory + RAG、Context Engineering、MCP / A2A / ANP 通訊協定、Agentic RL、評測
- **案例篇** — 智慧旅遊助理、Deep Research Agent、賽博小鎮多代理模擬
- **大作業** — 學習者基於 OpenAI 原生 API 自建 `HelloAgents` 框架

### 目標使用者
具 Python 與 LLM 基礎的 AI 開發者、軟體工程師、學生、自學者；強調「動手實作」而非演算法深訓。

### 與類似專案的差異
教程明白區分「軟體工程式 Agent 平台（流程驅動，LLM 當後端）」與「AI-native Agent（真正由 AI 驅動）」，後者才是主軸——這跟同社群的 [datawhalechina/llm-cookbook](https://github.com/datawhalechina/llm-cookbook)（偏 LLM 應用）、[microsoft/AI-Agents-for-Beginners](https://github.com/microsoft/AI-Agents-for-Beginners)（微軟版 10 課入門）相比，章數更多、覆蓋 Agentic RL 與通訊協定，更接近一本完整中文 Agent 教科書。
