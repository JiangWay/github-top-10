## 每日一字

> **比肩（Shoulder to Shoulder）**

從總結中挑出「比肩」，是因為今日榜上**三組「並列／追平」結構**罕見地同時浮現：

1. **連榜紀錄比肩**：[1jehuang/jcode](https://github.com/1jehuang/jcode) 連 6 日上榜，**正式追平本站連榜紀錄保持人 [mattpocock/skills](https://github.com/mattpocock/skills) 的 6 日紀錄**——個人作者主導的 Rust agent harness 與企業背景的 Skill marketplace 在「能不能持續吸引 trending 演算法的目光」這件事上**比肩而立**。jcode 的差異是：mattpocock/skills 連 6 日依靠的是上游 release 推力，jcode 今日 release 端「靜止 24+ 小時」依然續增 stars——是更純粹的有機曝光。
2. **金融科技雙檔比肩**：[TauricResearch/TradingAgents](https://github.com/TauricResearch/TradingAgents)（連 5 日、絕對榜 #2）與 [virattt/dexter](https://github.com/virattt/dexter)（首次、絕對榜 #10）今日同框，**為金融科技賽道首見「連榜種子＋新進挑戰者」雙檔同日上榜**。前者主打「多代理 LLM 金融交易框架」走 Python；後者主打「住在終端機裡的金融研究 agent」走 TypeScript，技術路線不同但目標使用者高度重疊，下週兩者誰能持續守榜將是金融科技 LLM 工具的關鍵指標。
3. **Rust Agent 雙王比肩**：[Hmbown/DeepSeek-TUI](https://github.com/Hmbown/DeepSeek-TUI)（連 2 日、growth #1、33.91%）與 [1jehuang/jcode](https://github.com/1jehuang/jcode)（連 6 日、growth #3、14.23%）兩檔 Rust 寫的 coding agent TUI 連 3 日同場競技，今日合計貢獻 stars_today +1,822（佔當日全榜 18.4%）；前者押 DeepSeek 原生 + RLM 平行，後者押多 provider 通用 harness，**兩條技術路徑在「Rust 終端 agent」這一狹窄賽道上比肩而行**，是 5 月以來最穩定的雙王結構。

「比肩」兩個字描述的是**勢均力敵、並肩同行的競合態勢**——不是誰壓過誰，而是多組對手同時推到同一海拔線。今日 9,923 的 total stars_today 從谷底連 2 日反彈 +29.9% → +19.3%（合計 +54.9%），動能不是某個單一爆款，而是這幾組「比肩結構」同時加柴。
