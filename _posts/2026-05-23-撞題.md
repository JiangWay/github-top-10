## 每日一字

**撞題（Convergent Discovery）**

「撞題」指兩個獨立團隊在毫不知情下對同一問題交出不同形態的答卷——今日 #2 [colbymchenry/codegraph](https://github.com/colbymchenry/codegraph) 與 #7 [Lum1104/Understand-Anything](https://github.com/Lum1104/Understand-Anything) 正是這一場景的具體展開：兩檔皆「把 codebase 轉成知識圖譜」，但 codegraph 走 MCP server + tree-sitter + SQLite/FTS5 路線把節點壓縮給 agent 吃（token 降 94%），Understand-Anything 走 6-agent LLM pipeline 一次性生成 JSON 圖譜 + 教學型 web dashboard 給人看（dashboard 跑起來不再呼叫 LLM）。一個服務 agent、一個服務人，問題相同、答案互補——這正是「撞題」的真意。再加上同日 [odoo/odoo](https://github.com/odoo/odoo) 與 [trimstray/the-book-of-secret-knowledge](https://github.com/trimstray/the-book-of-secret-knowledge) 兩檔正典級長壽 repo 首登「重新撞回」本站視野，撞題之外更撞時。
